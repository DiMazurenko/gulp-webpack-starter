# gulp-webpack-starter

Gulp assembly for project HTML/Sass/JS

## Description
Convenient project assembly:
- from several html files
- using the sass preprocessor
- webpack
- font formatting
- image formatting
- zip archive with the finished project
- FTP project deployment

## Setup
```sh
npm install
```

### developer mode
```sh
npm run dev
```

### build mode
```sh
npm run build
```

### deploy FTP 
```sh
npm run deploy
```

### make ZIP  
```sh
npm run zip
```
